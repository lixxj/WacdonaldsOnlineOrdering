# Wacdonald Online Ordering System

## UNSW COMP1531 19T1 Assignment

### How to use:

    git clone "git@github.com:lixxj/Wacdonalds-Ordering-System.git" 
  
    cd Wacdonald-Order-System     
    
    python3 run.py
  
### flask and jinja2 are required, so optionally:
  
    pip3 install flask

    pip3 install jinja2
